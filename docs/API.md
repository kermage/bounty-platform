# API Documentation

## Health
- `GET /health`

## Bounties
- `GET /api/bounties`
- `POST /api/bounties`
- `GET /api/bounties/:id`
- `PUT /api/bounties/:id`

## Submissions
- `POST /api/bounties/:id/submissions`
- `GET /api/bounties/:id/submissions`

## Disputes
- `POST /api/bounties/:id/disputes`
- `GET /api/disputes/:id`
